# one--pagewebsite

A simple one-page website built with **HTML** (and optionally CSS/JS).

## Preview
- Open `index.html` in your browser, or enable GitHub Pages to host it.

## Getting started
1. Clone the repo:
   ```bash
   git clone https://github.com/safimahinja-cmd/one--pagewebsite.git
   cd one--pagewebsite
   ```
2. Open the site locally:
   - Double-click `index.html`, **or**
   - Use a local server (recommended):
     ```bash
     # Python 3
     python -m http.server 8000
     ```
     Then visit `http://localhost:8000`.

## Project structure
- `index.html` — main page
- `assets/` — images, CSS, JS (if present)
